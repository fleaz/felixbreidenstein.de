My projects

Most of my projects are hosted on Github or my private Gitlab instance.

CptHook Webhook to IRC Gateway with plugins for common tools like Gitlab and Prometheus

prometheus-storagebox-exporter A Prometheus exporter for Hetzner Storageboxes

r53-ddns A DynDNS client which uses AWS Route53

arduino-temp-exporter An Arduino Nano with a network shield and a temperature sensor, which you can include in your Prometheus monitoring

sqlstrip Remove tables from a sql dump to make it import faster

s3-ecouploader A sync tool for S3 with the goal to be time and memory efficient (Developed for a low-end NAS)

shared-2fa A CLI "password manager" which allows to share TOTP token in a team for shared accounts

nocontent A serverless application which renders placeholder images on demand for webdev prototyping
